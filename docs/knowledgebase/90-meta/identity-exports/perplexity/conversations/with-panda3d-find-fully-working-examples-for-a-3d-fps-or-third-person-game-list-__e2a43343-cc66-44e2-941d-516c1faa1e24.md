   - This project aims to create a flexible FPS framework using Panda3D. It demonstrates level handling, plugin architecture, and various effects. 3. **Naith**
   - Naith is another FPS demonstration using Panda3D. The project aims to provide a complete game framework but is currently stalled. It offers a playable environment with components like level handling and effects examples.
   - Source Code: [Naith](https://github.com/Panda3D-public-projects-archive/naith).[3]
