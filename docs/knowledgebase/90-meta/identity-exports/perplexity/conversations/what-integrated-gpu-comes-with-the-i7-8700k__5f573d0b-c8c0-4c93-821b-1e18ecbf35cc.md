# What integrated gpu comes with the i7-8700k

Source: [Perplexity thread](https://www.perplexity.ai/search/ecd19639-7ca6-4f6d-a866-9e5b7b2d7b2b)

## User

What integrated gpu comes with the i7-8700k

## Assistant

The Intel Core i7-8700K processor comes with an integrated GPU called Intel UHD Graphics 630. This integrated graphics solution is designed to handle basic display tasks and light graphics workloads.[1][2][4]

## Key Specifications

- **GPU Name**: Intel UHD Graphics 630
- **Base Frequency**: 350 MHz
- **Max Dynamic Frequency**: 1.20 GHz
- **Max Video Memory**: 64 GB
- **DirectX Support**: 12
- **OpenGL Support**: 4.5

## Display Capabilities

The UHD Graphics 630 integrated into the i7-8700K supports:

- **4K Resolution**: Yes, at 60Hz
- **Max Resolution (HDMI)**: 4096 x 2304 @ 24Hz
- **Max Resolution (DisplayPort)**: 4096 x 2304 @ 60Hz
- **Number of Displays Supported**: 3

## Performance Considerations

While the UHD Graphics 630 is capable of handling everyday tasks like web browsing, document processing, and video playback, it has limitations for gaming and graphics-intensive applications. For basic esports titles and less demanding games, it can provide playable performance at lower settings. However, for more demanding AAA games or professional graphics work, a dedicated GPU would be recommended.[3][6]

It's worth noting that some users have reported issues with driver installation or detection of the integrated graphics in certain system configurations. In such cases, ensuring proper BIOS settings and up-to-date drivers from the motherboard manufacturer may be necessary.[5]
