# operagx enable vertical tabs

Source: [Perplexity thread](https://www.perplexity.ai/search/ee10f1bb-3a0d-4079-b032-f331eff5948f)

## User

operagx enable vertical tabs

## Assistant

To enable vertical tabs in Opera GX, you'll need to use a browser extension as this feature is not natively built into the browser. Here's how you can add vertical tabs to Opera GX:

## Installing a Vertical Tabs Extension

1. Open the Opera add-ons store in your Opera GX browser.

2. Search for "Vertical Tabs" in the store.

3. Look for the "Vertical Tabs" extension by the developer "Crx4." This is a popular and well-maintained extension for vertical tabs.[1][2]

4. Click on the "+ Add to Opera" button to install the extension.

## Setting Up Vertical Tabs

Once the extension is installed, you'll need to activate and configure it:

1. Ensure that the Opera sidebar is activated. If it's not visible, press Ctrl + Shift + S to open it.

2. Look for the Vertical Tabs icon in the sidebar and click on it to activate the vertical tabs view.

3. Your open tabs will now be listed vertically on the left side of the browser window.

## Customizing Vertical Tabs

The Vertical Tabs extension offers several customization options:

1. **Tab Management**: 
   - Click on tabs in the vertical bar to switch between them.
   - Close tabs by clicking the "X" next to each tab.
   - Open new tabs using the "+" button at the top of the sidebar.

2. **Advanced Features**:
   - Search tabs using the "Search tabs" function at the top of the sidebar.
   - Sort tabs alphabetically with the "Sort tabs" button.
   - Create tab groups for better organization.[1]

3. **Settings**:
   - Click the spanner icon at the top of the sidebar.
   - Select "Vertical tabs settings" to access more options.
   - Here you can change the theme, customize toolbars, and adjust other preferences.[1]

## Additional Tips

- You can use Ctrl+click to select multiple tabs, or Shift+click to select a range of tabs.[2]
- Double-click on empty space in the tab list to create a new tab.[2]
- Hover over the tab list and press Ctrl+A to select all tabs, or Ctrl+I to invert your selection.[2]

By following these steps, you'll be able to enjoy a vertical tab layout in Opera GX, which can be particularly useful for managing many open tabs efficiently, especially on widescreen monitors.
