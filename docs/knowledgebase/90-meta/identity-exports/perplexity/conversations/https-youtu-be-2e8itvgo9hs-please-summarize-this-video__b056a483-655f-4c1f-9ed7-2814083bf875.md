## What is Langfuse?

Langfuse is an open-source LLM engineering platform designed for observability, evaluation, and prompt management of LLM applications. It's primarily an observability layer that works asynchronously, meaning it doesn't sit between your application and the LLM provider's API, but rather observes and logs interactions "from the sidelines".[1][3][4]

## Key Functions

1. **Observability**: Langfuse allows comprehensive tracing and monitoring of LLM applications, capturing all relevant data points and interactions.[4]

2. **Prompt Management**: It offers tools to manage, version, and deploy prompts efficiently.[4]

3. **Evaluation and Metrics**: Langfuse supports various evaluation mechanisms, including user feedback, model-based evaluations, and manual scoring.[4]

4. **Analytics**: It provides dashboards and data exports for tracking metrics such as cost, latency, and quality.[1]

5. **Experimentation and Testing**: Langfuse enables running experiments and tracking application behavior before deploying new versions.[4]

## Comparison with LiteLLM and Model Context Protocol

- Unlike LiteLLM, which is a proxy for standardizing LLM APIs, Langfuse is not a proxy but an observability platform.[3]

- While the Model Context Protocol aims to standardize how AI applications interact and exchange data, Langfuse focuses on monitoring, analyzing, and optimizing existing LLM applications.[6]

In summary, Langfuse is a comprehensive tool for monitoring, debugging, and optimizing LLM workflows, rather than a protocol for standardizing AI interactions or a proxy for LLM APIs.

## Letta Overview

Letta is an open-source framework for building stateful LLM (Large Language Model) applications. It adds state and memory capabilities to LLMs, enabling advanced reasoning and transparent long-term memory for AI agents.

## Letta ADE (Agent Development Environment)

The Letta ADE is a web-based graphical user interface for creating, deploying, interacting with, and observing Letta agents. Key features of the ADE include:

1. Agent Creation: Allows customization of starter data for the agent's memory and persona.
2. Real-time Interaction: Enables messaging the agent and observing its responses, including inner thoughts and reasoning.
3. Memory Visualization: Displays both core (in-context) and archival (out-of-context) memories of the agent.
4. Tool Management: Shows available tools, including memory tools and default Letta tools.
5. Development Controls: Provides toggles to control what is exposed to end-users and starter messages for agent triggers.
