## Assistant

Here are detailed instructions for setting up Mantella for Skyrim:

1. Download and install Mantella using your preferred mod manager (Mod Organizer 2 or Vortex).

2. Locate the Mantella mod folder:
   - For Mod Organizer 2: Right-click the Mantella mod in the UI and select "Open in Explorer"
   - For Vortex: Navigate to "\Data\Mantella" in your Skyrim game folder (e.g., C:\Games\Steam\steamapps\common\Skyrim Special Edition\Data\Mantella)

3. Create a free account with OpenRouter. Go to the "Keys" tab and generate a new key.

4. Save the OpenRouter key to the "GPT_SECRET_KEY.txt" file in your Mantella folder.

5. Install required Skyrim mods:
   - Skyrim Script Extender (SKSE)
   - Address Library for SKSE Plugins
   - VoiceActivityDetection (or Fuz Ro D-oh - Silent Voice for lip sync)
   - MCM Helper (for accessing Mantella's configuration menu)

6. Install optional but recommended mods:
   - No NPC Greetings (to prevent interruptions during Mantella conversations)
   - Mantella - USSEP Patch (if using Unofficial Skyrim Special Edition Patch)

7. Ensure Mantella is loaded after USSEP in your mod load order.

8. Launch Skyrim through SKSE.

9. When Skyrim launches, Mantella will open a window automatically. It may take a few minutes on first launch.

10. The Mantella spell will be added to your inventory under the Illusion category in the Magic menu.

11. To start a conversation, cast the Mantella spell on an NPC.

12. Access Mantella's configuration menu in-game via System > Mod Configuration > Mantella to adjust settings.

13. For group conversations, cast Mantella on additional NPCs between your responses.

14. End conversations by casting the Mantella End Conversation spell or by saying/typing "goodbye".

Remember to check for compatibility with other mods and ensure you're using the correct versions for your Skyrim edition (SE/AE/VR). If you encounter any issues, consult the mod's Discord community for troubleshooting assistance.[1][2][3][4]
