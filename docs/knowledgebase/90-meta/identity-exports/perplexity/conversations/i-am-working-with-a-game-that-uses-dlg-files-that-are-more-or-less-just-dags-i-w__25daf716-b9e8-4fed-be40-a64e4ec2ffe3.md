Based on these, here are the top 3 repositories I'd recommend checking out:

1. ClearDialogue (https://github.com/SkyAphid/ClearDialogue)
   This is an open source, flexible dialogue editor that supports JSON export/import. It's designed to be engine-agnostic and highly customizable. The code is well-documented and it has features like syntax highlighting and find/replace functionality.

2. Dialogue Nodes (https://github.com/nagidev/DialogueNodes)
   This is a plugin for Godot that allows creating branching dialogues directly within the engine. It supports conditional dialogues, variables, and localization. While specific to Godot, its implementation could provide useful insights.

3. DlgSystem (https://github.com/NotYetGames/DlgSystem)
   This is a dialogue plugin for Unreal Engine with both Blueprint and C++ support. It has a graph editor, search functionality, and supports importing/exporting to plain text formats. Its structure and features could be valuable references even if you're not using Unreal.
