## Assistant

Apple added support for Rich Communication Services (RCS) in **September 2024** with the release of **iOS 18**. This update significantly improved messaging between iPhone and Android users, addressing long-standing issues with cross-platform communication. Before this, messages between iPhones and Android devices were limited to SMS/MMS, which lacked many modern features like high-resolution media sharing, read receipts, and typing indicators.

The introduction of RCS in iOS 18 allows for a more seamless experience between iPhone and Android users, enabling features such as:

- High-resolution photo and video sharing
- Larger file sizes
- Cross-platform emoji reactions
- Real-time typing indicators
- Read receipts
- The ability to send messages over both cellular and Wi-Fi

This marked a significant step forward in bridging the messaging gap between iOS and Android devices.[4][5]
