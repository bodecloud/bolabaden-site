# in perplexity.ai, is it possible to copy the the entire conversation or save it locally? All of the copy/share buttons are links to the conversation on the website. How to locally download the conversations?

Source: [Perplexity thread](https://www.perplexity.ai/search/53cd61fd-b750-4d3a-908e-db358e6cf082)

## User

in perplexity.ai, is it possible to copy the the entire conversation or save it locally? All of the copy/share buttons are links to the conversation on the website. How to locally download the conversations?

## Assistant

As of November 2024, Perplexity AI does not have a native feature to directly copy or download entire conversations locally. However, there are third-party solutions available to help you save Perplexity conversations:

1. Save my Chatbot Extension: This is a popular browser extension specifically designed to export conversations from Perplexity AI (and other AI chatbots) into markdown files. Here's how it works:[1][2]

   - Install the "Save my Chatbot" extension from the Chrome Web Store or Firefox Add-ons.
   - Navigate to the Perplexity conversation you want to save.
   - Click on the extension icon in your browser.
   - The extension will automatically download a structured markdown file containing the conversation.

2. Manual Copy-Paste: While not as convenient, you can manually copy and paste the conversation text into a document or text file of your choice.

3. Browser Print/Save as PDF: You can use your browser's print function (usually Ctrl+P or Cmd+P) and choose "Save as PDF" to create a PDF copy of the conversation.

4. Screenshot Tools: For visual preservation, you can use screenshot tools or browser extensions to capture the entire conversation page.

It's worth noting that Perplexity's interface and features may change over time, so it's always a good idea to check their official documentation or support channels for the most up-to-date information on saving or exporting conversations.
