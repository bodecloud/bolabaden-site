Yes, Spyder IDE is still actively used and developed in 2024. The recent release of Spyder 6 in September 2024 brought several significant updates and new features to the IDE. Here are some of the highlights from Spyder in 2024:

## Major New Features

1. **Remote Development Support**: Spyder 6 introduced initial support for connecting to remote workstations, cloud desktops, and computing clusters via SSH. This allows users to install and start kernels remotely, as well as run code on remote machines.[3]

2. **Conda-based Installers**: New standalone installers for Windows, macOS, and Linux were introduced, offering more convenient and robust updates, newer Python versions, and paving the way for future features.[3]

3. **Debugger Pane**: A new Debugger pane was added to explore the stack frame of the current debugging session, centralizing debugger functionality in one place.[3]

## Global Enhancements

- **Plugin Management**: Users can now manage external plugins (e.g., Spyder-Notebook) in the Preferences > Plugins section.[3]
- **Configurable Interface Font**: The application's interface font is now customizable.[3]
- **Improved Input Support**: Added support for Chinese, Japanese, and Korean input on Linux.[3]

## Editor and Files Improvements

- **Quick Switcher**: The Ctrl/Cmd-P shortcut now allows browsing and opening files present in the current project.[3]
- **File Path Copying**: Users can now copy the absolute and relative paths of the current file in the Editor tab context menu.[3]

## IPython Console Updates

- **Environment Integration**: New submenu to start a console in any registered Conda or Pyenv environment with one click.[3]
- **Faster Kernel Restarts**: Kernel restarts are now much quicker.[3]

## Variable Explorer Enhancements

- **Refresh Button**: Added a new refresh button for viewer windows to update variables within them.[3]
- **Improved Data Type Support**: All real number data types are now supported in the DataFrame viewer.[3]

## Community Reception

The Spyder 6 release has been well-received by the community. Within a month and a half of its release, the new standalone installers for Spyder 6.0.0 and 6.0.1 were downloaded over 350,000 times across Windows, Mac, and Linux platforms. Users on platforms like Reddit have expressed enthusiasm for the new features and improvements.[5][3]

## Future Developments

The Spyder team has announced plans for future enhancements, including:

- Further improvements to the remote development platform.[3]
- A built-in package and environment manager for creating Conda environments linked to projects.[3]
- A graphical Spyder plugin browser for easy installation and management of external plugins.[3]

In conclusion, Spyder IDE remains a popular choice for scientists and data analysts in 2024, with its latest version bringing significant improvements in usability, performance, and feature set.
