## **Sparse Distributed Memory (SDM)**

1. **"Sparse Distributed Memory" by Pentti Kanerva (1988)**: This foundational work introduces SDM as a model of human long-term memory. SDM is based on high-dimensional spaces and is designed to store and retrieve large amounts of information based on similarity rather than exact matches. The key feature of SDM is its robustness to partial damage in input data, which allows it to retrieve correct outputs even when inputs are noisy or incomplete. This makes it a powerful tool for tasks like pattern recognition and associative memory. The Wikipedia article on SDM provides a concise summary of these concepts, highlighting how SDM mimics human memory processes by mapping logical addresses to physical locations in a distributed manner.[2]

2. **"In-memory Computing Architectures for Sparse Distributed Memory" by Mingu Kang et al.**: This paper presents an advanced architecture for implementing SDM using in-memory computing techniques. It discusses how SDM can be efficiently implemented in hardware through energy-efficient architectures, which could be particularly useful if you're interested in hardware implementations of memory models for machine learning applications. The paper also explores how these architectures can handle noisy or incomplete data, much like the human brain.[5]

3. **"Distributed-Memory Sparse Kernels for Machine Learning" by Aydin Buluc et al.**: This paper focuses on sparse matrix operations, such as Sampled Dense Times Dense Matrix Multiplication (SDDMM) and Sparse Times Dense Matrix Multiplication (SpMM), which are important for applications like graph embeddings and collaborative filtering. While not directly about SDM, the discussion on sparse representations and their use in machine learning could provide insights into how sparse distributed representations are applied in modern computational settings.[1]

## **Regular Equivalence in Network Embedding (RSA)**

1. **"Deep Recursive Network Embedding with Regular Equivalence" by Ke Tu et al.**: This paper introduces a novel method called Deep Recursive Network Embedding (DRNE), which uses regular equivalence to learn network embeddings. Regular equivalence is a concept from graph theory where nodes that play similar roles in a network (even if they are not directly connected) receive similar embeddings. This approach is particularly useful for tasks like structural role classification in networks. The paper demonstrates how DRNE outperforms other methods by preserving both local and global structural information in networks.[3]

2. **"Study and Analysis of Routing and Spectrum Allocation (RSA) Algorithms in Elastic Optical Networks"**: Although this paper focuses on routing and spectrum allocation within optical networks, it provides insights into RSA algorithms used in network optimization tasks. Understanding RSA from this perspective can help you see how embedding techniques are applied to optimize complex network systems, potentially offering analogies for embedding tasks in machine learning.[4]

These readings should provide you with a solid foundation on both SDM and RSA, helping you explore alternative embedding methods beyond transformer encodings.
