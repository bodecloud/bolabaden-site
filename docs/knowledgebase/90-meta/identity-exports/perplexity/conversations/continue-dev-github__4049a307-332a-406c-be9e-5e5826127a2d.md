## Assistant

**Continue.dev** is an open-source AI code assistant designed to enhance developer productivity by integrating AI-based features directly into the coding environment. It allows users to connect various language models (LLMs) and customize autocomplete and chat experiences inside their IDEs, such as Visual Studio Code and JetBrains.[1][3]

### Key Features:
- **Autocomplete**: Continue can autocomplete single lines or entire sections of code in any programming language as you type.
- **Code Reference and Chat**: You can reference parts of your codebase and ask questions about functions, files, or even the entire project.
- **Customizable**: Continue allows developers to integrate their own models or use API keys for LLM providers like Anthropic. You can also customize the context providers to reference files, Git diffs, terminal outputs, and more.[3][6]
- **Free Trial**: New users can try Continue with a free trial using API keys provided by Continue. After the trial, users need to either select an open-source model or provide their own API key.[4]

Continue is available for free and can be used with local models like Ollama or other major LLM providers, making it a popular alternative to paid tools like GitHub Copilot.[7]
