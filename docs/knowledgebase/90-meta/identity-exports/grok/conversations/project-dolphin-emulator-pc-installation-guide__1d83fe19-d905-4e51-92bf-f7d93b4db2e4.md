### Step 2: Extract the Files
1. Download and install **7-Zip** (free) or **WinRAR** if you don’t have a zip extractor.
2. Right-click the downloaded Project+ zip file → **Extract Here** or **Extract to folder**.
3. You should now have a folder with:
   - `Dolphin.exe` (or similar launcher)
   - A `Games` folder
   - Other files/folders

### Step 3: Place Your Brawl ISO
1. Copy your **Super Smash Bros. Brawl ISO** file (usually ends in `.iso`).
2. Paste it into the **Games** folder inside your Project+ folder.<grok:render card_id="994b25" card_type="citation_card" type="render_inline_citation"><argument name="citation_id">59</argument></grok:render>

### Step 4: Run Project+
1. Double-click **Dolphin.exe** (or the launcher file) in the main Project+ folder.
2. The first time it runs, it may take a minute to configure.
3. Your game should appear in the list as **Project+** or **Super Smash Bros. Brawl** with the mod applied.
4. Double-click the game to start playing!

### Step 5: Basic Settings (Important for Performance)
1. In Dolphin, go to **Options → Graphics Settings**:
   - Set **Backend** to **Vulkan** (best) or **Direct3D 11**.
   - Resolution: 1080p or 720p.
   - Enable **Anti-Aliasing** (2x or 4x) if your PC can handle it.
2. Go to **Config → General** and make sure paths are set correctly.
3. **Controller Setup**:
   - Go to **Controllers**.
   - Set up your keyboard, GameCube adapter, or controllers (Xbox, PlayStation, etc.).
   - Standard layout works great for Smash.

### Step 6: Common Fixes for Issues
- **Game doesn’t show up?** Right-click in Dolphin → Refresh List. Make sure the ISO is in the Games folder.
- **Black screen or crashes?** Update your graphics drivers. Try lowering graphics settings.
- **Slow performance?** Close background programs. Use Vulkan backend. Lower resolution.
- **No sound or glitches?** Restart Dolphin or check Audio settings.
