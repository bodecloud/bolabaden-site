### Enabling the Console: Step-by-Step Guide
The console in KOTOR 1 allows access to cheats like adding items, levels, or credits, primarily for PC versions (Steam, GOG, etc.). #### Detailed PC Instructions
1. **Locate the file**: Right-click KOTOR in Steam > Manage > Browse local files. Path: `...\swkotor\swkotor.ini`. (Non-Steam: `C:\Program Files (x86)\LucasArts\SWKotOR\swkotor.ini` or similar.)<grok:render card_id="bcc893" card_type="citation_card" type="render_inline_citation">
<argument name="citation_id">72</argument>
</grok:render><grok:render card_id="7125c8" card_type="citation_card" type="render_inline_citation">
<argument name="citation_id">76</argument>
</grok:render>
2. **Edit safely**: Right-click > Open with Notepad (run as admin if needed). Find `[Game Options]` (Ctrl+F).
3. **Add line**: Below it, type exactly: `EnableCheats=1` (case-sensitive, no spaces/quotes). Save (Ctrl+S).<grok:render card_id="d1a08f" card_type="citation_card" type="render_inline_citation">
<argument name="citation_id">72</argument>
</grok:render><grok:render card_id="7d0cd3" card_type="citation_card" type="render_inline_citation">
<argument name="citation_id">75</argument>
</grok:render>
4. **Launch and test**: Load a save. Press `~`—a `>` prompt appears top-left. Type a cheat (e.g., `heal`), Enter. Hold Tab for auto-complete list.<grok:render card_id="c53900" card_type="citation_card" type="render_inline_citation">
