The GUI in *Star Wars: Knights of the Old Republic II - The Sith Lords* (KOTOR 2 TSL) features a distinctive color scheme that emphasizes functionality within a immersive Star Wars universe, drawing from the game's themes of moral ambiguity, ancient technology, and galactic conflict. This comprehensive survey compiles color data from modding communities, fan analyses, and game asset extractions, focusing primarily on GUI elements while extending to broader palettes that "match" the game's visual style. All colors are presented in HEX format for practical use, with conversions from normalized RGB where applicable. Note that game colors can appear differently based on hardware, lighting, or mods like TSLRCM (The Sith Lords Restored Content Mod), but these represent the most substantiated values from reliable sources.

#### Core GUI Color Analysis
The GUI in KOTOR 2 is built around a green-dominated palette, which modders often describe as evoking a holographic or terminal-like interface. This choice aligns with the game's narrative of a fractured Republic and Sith remnants, using cool, subdued tones to convey isolation. Colors are stored in .gui files (GFF format) and editable via tools like KotOR GUI Editor. Default values are normalized RGB (0-1 scale), which we've converted to HEX for accessibility.

**Table 1: Default GUI Colors (From Modding Resources)**
| Color Category | Name          | Normalized RGB             | HEX Code   | Usage Examples                     |
|----------------|---------------|----------------------------|------------|------------------------------------|
| Green Scheme  | Dark Green   | 0.050980, 0.349020, 0.270588 | #0C5944   | Menu borders, shadows, and accents in inventory, map, and dialogue screens. Provides depth without clashing with backgrounds. |
| Green Scheme  | Light Green  | 0.101961, 0.698039, 0.549020 | #1AB18C   | Highlights, selectable buttons, and active elements like health bars or skill icons. Creates a glowing effect. |
| Green Scheme  | Text Green   | 0.101961, 0.698039, 0.549020 | #1AB18C   | Default text for labels, descriptions, and options. Ensures readability against darker backgrounds. |
| Neutral Grays | Very Light Grey | 0.9098039, 0.9098039, 0.9098039 | #E7E7E7 | Subtle backgrounds, tooltips, or inactive areas. Often used in templates for custom mods. |
| Neutral Grays | Light Grey   | 0.7490196, 0.7490196, 0.7490196 | #BEBEBE   | Mid-tone panels, dividers, or unselected tabs. Balances the green dominance. |
| Neutral Grays | Dark Grey    | 0.4901961, 0.4901961, 0.4901961 | #7D7D7D   | Deeper shadows, borders in low-light scenes, or error indicators. |

These values stem from direct examinations of game files via modding forums, where users discuss recoloring for personalization (e.g., to blue for a KOTOR 1-like feel). No official HEX documentation exists from Obsidian or LucasArts, but community consensus holds these as accurate. For instance, mods like "Blue Kotor-Like GUI" automate color shifts but retain these as base references, adjusting greens to blues without listing specifics. If editing, divide desired RGB by 255 to get normalized values for .gui files.
