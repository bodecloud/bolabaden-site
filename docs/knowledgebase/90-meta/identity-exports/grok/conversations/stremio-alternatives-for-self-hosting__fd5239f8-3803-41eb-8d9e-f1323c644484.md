##### Self-Hosted Alternatives for Torrent Streaming
Given the lack of direct forks, we focused on self-hosted projects that offer similar torrent streaming capabilities. Here are the key findings:

- **TorrServer**: Found at [TorrServer GitHub Repository](https://github.com/YouROK/TorrServer), this is a torrent stream server that allows viewing torrents online without downloading first. It caches torrents and streams via HTTP, with a web interface accessible at http://127.0.0.1:8090 after installation. Its latest release, MatriX.135, was on April 25, 2024, indicating active maintenance. Features include support for various platforms (Windows, Linux, Mac, Raspberry Pi), DLNA/UPnP support, and a dedicated Android client (TorrServe). - **Rapid Bay**: Located at [Rapid Bay GitHub Repository](https://github.com/hauxir/rapidbay), this is a self-hosted video service/torrent client compatible with Chromecast, AppleTV, and Kodi. However, its last commit was on February 23, 2023, suggesting it may not be actively maintained, which could pose risks for long-term use.

- **Webtor (Self-Hosted)**: Found at [Webtor Self-Hosted GitHub Repository](https://github.com/webtor-io/self-hosted), this is a web-based torrent client with instant audio/video streaming, implemented as a Docker image. It supports direct download links, streaming of video (avi, mkv, mp4, webm, m4v, ts, vob) and audio (mp3, wav, ogg, flac, m4a), and downloading entire torrents as ZIP archives. Its maintenance status is unclear, with last updates around 2023, but it’s developer-friendly with an SDK for website integration.
