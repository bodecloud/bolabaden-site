#### Definition of Neurons
Neurons are Cloudflare's proprietary unit for measuring AI compute usage across different models on their Workers AI platform. They represent the GPU computational effort required to process a request, abstracting the complexity of varying model sizes and input/output requirements into a single metric. This approach allows Cloudflare to standardize pricing and usage tracking across diverse AI tasks, such as text generation, image processing, and embeddings, without requiring users to manage GPU resources directly. Neurons are not tied to a specific action like "per message" but rather aggregate the compute cost of processing inputs and outputs for a given model.[](https://developers.cloudflare.com/workers-ai/platform/pricing/)[](https://developers.cloudflare.com/workers-ai/platform/pricing/index.md)

#### Neurons and "Per Day" Allocation
The free tier's allocation of 10,000 Neurons per day is a cumulative limit that resets daily at 00:00 UTC. This means all API requests made within a 24-hour period consume Neurons based on the model used and the input/output processed, up to a total of 10,000 Neurons. Once this limit is exceeded, further operations fail unless the user upgrades to the Workers Paid plan, where additional Neurons are billed at $0.011 per 1,000 Neurons. There are no charges per message within the free tier, as long as the total Neuron consumption stays within the 10,000 daily limit.[](https://developers.cloudflare.com/workers-ai/platform/pricing/)

#### Neurons in Relation to Messages
Neurons are not directly equivalent to "messages" or individual API requests. Instead, the Neuron cost of a request depends on factors such as:
- **Model Type and Size**: Larger models (e.g., Llama-3.1-70b) consume more Neurons per token than smaller ones (e.g., Llama-3.2-1b).
- **Input and Output Tokens**: The number of tokens in the input prompt and generated output affects Neuron usage, with output tokens often consuming more Neurons due to higher computational demands.
- **Task Complexity**: Different tasks (e.g., text generation vs. embeddings) have varying Neuron costs.

For example, a single API request sending a prompt to an LLM and receiving a response is considered one "message" in typical AI terminology. The Neuron cost for this message is calculated based on the tokens processed and the model’s pricing in Neurons per token. Multiple messages (e.g., a conversation with several prompts and responses) accumulate Neuron usage toward the daily limit.[](https://developers.cloudflare.com/workers-ai/platform/pricing/index.md)
