### Comprehensive Overview of New Planet Mods for KOTOR 2: TSL
KOTOR 2: The Sith Lords (TSL) has a vibrant modding community, primarily centered on Deadly Stream (the modern hub for starwarsknights.com forums), Nexus Mods, and ModDB. Mods adding truly new planets—via a planetary.2da entry for galaxy map visibility, excluding base game content (Peragus, Telos, Nar Shaddaa, Dxun/Onderon, Dantooine, Korriban, Malachor)—are rare due to technical limits (16 slots max) and development effort. Extensive searches (queries targeting "new planet," "planetary.2da," site-specific on Deadly Stream/Nexus/Reddit/LucasForums Archive, excluding knowns iteratively) confirmed only three qualifying mods or suites.

#### Installation Best Practices
1. Backup saves/overrides.
2. Install TSLRCM 1.8.6 first.
3. Run TSLPatchers in order: Expanded/Coruscant → Cathalan.
4. Use pre-Telos save for galaxy map updates.
5. Test compatibility; patches available for overlaps.

These mods significantly extend playtime (2+ hours per planet) while preserving TSL's feel.
